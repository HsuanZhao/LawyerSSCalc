# 提成律师社保实缴计算器

> 一个为提成律师量身定制的交互式社保与到手收入测算工具，直观展示“按最低基数缴纳”与“按实际收入 100% 实缴”之间的巨大差异。

## ✨ 功能特点

- **双向对比**：并列显示“最低基数”与“实际收入基数”下的费用拆解（管理费、双边社保、个税、到手收入）。
- **实时可视化**：柱状图与数据标签同步更新，金额占比一目了然。
- **城市预设**：内置上海、北京、广州、深圳、杭州、成都等城市缴费基数上下限，一键切换。
- **全参数可调**：年创收、管理费率、社保费率（单位/个人）、基数上下限、专项附加扣除等全部开放自定义。
- **一键生成卡片**：将测算结果导出为高清 PNG 图片，方便分享或存档（支持下载与系统分享）。

## 🎯 适用人群

- 提成制律师（尤其是独立执业、自负社保的律师）
- 律所财务或行政人员，用于测算不同缴费方案对律师实际收入的影响
- 关注社保负担与税后收入的法律从业者

## 🚀 使用方法

1. 打开 `提成律师社保实缴计算器.html` 文件（浏览器即可运行，无需网络）。
2. 在“基础参数”区域输入：
   - **年创收**（元）
   - **律所管理费率**（%）
   - 选择或自定义 **城市/口径预设**
3. 在“社保缴费基数”区域：
   - “之前·最低基数”通常为当地社保下限；
   - “之后·实缴基数”会根据创收与管理费自动估算，也可手动覆盖。
4. 调整“社保费率”与“个税扣除项”（系统已内置七级累进税率表）。
5. 右侧图表与下方汇总区将同步更新，展示：
   - 各费用项目金额及占比；
   - 社保多缴金额与到手收入变化；
   - 创收到手率的前后对比。
6. 点击 **“生成卡片并下载”** 或 **“分享卡片”** 导出结果图片。

## 📊 数据口径说明

- **社保缴费**：按“单位部分 + 个人部分”双边合计费率计算（提成律师需独自承担全部费用）。
- **个人所得税**：采用 2019 年后综合所得七级超额累进税率（年度预扣预缴口径），扣除项目包括：
  - 基本减除费用（年 60,000 元）；
  - 个人承担社保部分；
  - 其他专项附加扣除（可手动填写）。
- **管理费**：直接从创收中按比例扣除，不参与社保基数计算。
- **基数自动测算**：实缴基数 = (创收 × (1 – 管理费率)) / 12，并自动限制在上下限范围内。

> ⚠️ 本工具为估算模型，实际缴纳金额以当地社保、税务部门最新政策及律所内部规定为准。未计入公积金、年终奖、办案成本等因素。

## 🛠️ 技术实现

- 纯 HTML + CSS + JavaScript，无外部依赖（除用于导出的 `html2canvas` CDN 库）。
- 响应式设计，适配桌面与移动端。
- 所有计算逻辑在前端完成，数据不上传服务器，保障隐私安全。
